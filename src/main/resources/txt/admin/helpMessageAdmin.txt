🤖 Vocabu Bot - Complete Guide

🌟 Main Features:
• Powered by Google Translate
• Smart caching for faster responses

📝 How to use:
Send any text in Ukrainian or in English
Get instant English or Ukrainian translation

⚡ Commands:
/start - Welcome message
/help - This help guide
/status - Check Google Translate status
/stats - Translation statistics
/clear - Clear translation cache

📏 Limits:
• Max 5000 characters per message
• Rate limits apply (Google's policy)

🎯 Examples:
• "Привіт світ" → "Hi world"
• "Hello world" → "Привіт світ"
